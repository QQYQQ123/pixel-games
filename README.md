# 🎮 PixelPlay 像素游戏集合

欢迎来到 PixelPlay！这是一个包含多个经典小游戏的 HTML5 游戏集合。

## 🎯 游戏列表

| 游戏 | 说明 |
|------|------|
| 2048 | 经典 2048 滑动拼图游戏 |
| breakout | 像素打砖块 |
| memory | 记忆翻牌 |
| guess | 猜数字 |
| shooter | 打飞机 |
| wheel | 幸运转盘 |
| lottery | 彩票机选 |
| avatar | 像素头像生成器 |
| meme | 表情包生成器 |
| pixelart | 像素艺术画板 |
| salary | 工资计算器 |

## 🌐 在线游玩

直接在浏览器中打开 index.html 即可游玩所有游戏！

## 📦 技术栈

- 纯 HTML5 + CSS3 + JavaScript
- Canvas 2D 渲染
- LocalStorage 本地存储
