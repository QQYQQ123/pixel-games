# 打砖块开发记录

## 2026-04-10

### 性能问题修复（第2次）
- **用户反馈**：连锁爆炸还在，还会卡
- **根本原因**：砖块太多（360个）+ shadowBlur 发光效果 + 粒子效果
- **解决方案**：
  1. 砖块数量：18×20 = 360个 → **12×10 = 120个**
  2. 完全删除 shadowBlur（最耗性能的发光效果）
  3. 完全删除粒子系统（createParticle、particles数组、粒子更新、粒子绘制）
  4. 删除背景星星动画
  5. 道具掉落率从20%降到15%
  6. 穿透状态8秒（480帧）
- **文件大小**：32157字节 → **25097字节**（减少22%）
