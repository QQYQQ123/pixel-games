## 字体规范
- **最小字号 12px**，辅助文字可 11px，禁止出现 10px 及以下
- 导航链接 12px+
- 页面副标题 13px+
- 卡片内文字 12px+
- 工具/预设按钮 12px+
- 历史记录时间 11px（最小）
- Canvas 内文字按选项数量动态：≤6项用13px，≤10项用11px（可调大）

## 推箱子布局重构
- **问题**：canvas 总是不够大，关卡格子少时更明显
- **根因**：旧逻辑用 `window.innerHeight - 固定值` 算 `availH`，窗口实际可用高度 ≠ `window.innerHeight`
- **解决方案**：
  1. `body { display:flex; flex-direction:column; height:100vh; }`
  2. `.navbar { position:fixed }` 不参与 flex 布局
  3. `.game-area { flex:1; height:calc(100vh - 60px - 48px); }` 撑满中间
  4. `.level-select { height:auto; flex-shrink:0 }` 底部固定
  5. JS 用 `document.querySelector('.game-area').clientHeight` 精确计算 canvas scale
- **文件**：sokoban.html
